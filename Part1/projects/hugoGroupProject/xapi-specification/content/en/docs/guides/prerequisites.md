Complete this page before the live session or before you work through the steps alone.

## Technical requirements

| Area | What you need |
| ---- | ------------- |
| Browser | Recent Chrome, Firefox, Edge, or similar with JavaScript enabled |
| Network | Stable internet for the site, Twine stories, and LRS checks |
| Device | Laptop recommended; phone works for reading, not ideal for Twine |
| Account | Email sign-in for the workshop questionnaire (one submission per account when the form is published) |

No LRS credentials are required in the browser; statements go through the site proxy.

## Knowledge

- Basic HTTP and JSON
- Comfortable following step-by-step technical instructions

Deep xAPI expertise is not required; the guides and stories introduce what you need.
